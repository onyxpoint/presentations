Basic Automated System

  • Deploy all software using Puppet
  • Runs at a given schedule
  • Use the native package manager if possible!
  • Use MCollective to disable your automated schedule
  • Pros
    • Consistent testing and deployment automation
    • Easy to monitor (single point of reports, etc...)
  • Cons
    • May need to adjust schedules for system downtime
    • Regular schedules may need to be disabled for certain software
